# Coffee Shop App

A beautiful, functional, and user-friendly Coffee Shop React Native App. This app allows users to browse different coffee types, view their descriptions and prices, add them to a cart, and track their order history.

## Features
- Browse coffees and beans
- Detailed view for items with size/price selections
- Add items to Cart
- Favorites screen to save favorite items
- Order History
- State management with Zustand

## Installation
```bash
npm install
```

## Run the Application

### iOS
```bash
npx react-native run-ios
```

### Android
```bash
npx react-native run-android
```

## Testing
Run the Jest tests:
```bash
npm test
```
